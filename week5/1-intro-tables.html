Intro to Tables

Tables are away to display "charts" of data in HTML form.

Parts of the table

  • <table> - tag - defines the start and stop of a table.
  • <tr> - tag - table row defines the start and stop of a table row.
  • <td> - tag - table data defines the start and stop of a table data (cell).
