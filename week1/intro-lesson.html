Intro to Block-level Elements

Block level elements will place content on their own line.

Paragraph Tag

The paragraph tag is used to define "normal" content like paragraphs of text.

So any text that does not have an other tag type use a paragraph.

Headline Tag

The headline tag is used to define sections of an article. There are six versions of the headline tag.

Break Tag

The break tag is used to add a line break within a block-level element. It will return but not add a space above or below.
