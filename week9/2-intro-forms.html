Intro to Forms

Form Tag

Is used to define the form and tell it how and where to send the data.

  • action - where we send the data. -Required
  • method - get or post - get in url, post hidden
  • name or id - used to name the one form from another.

Input Tag

Input tag needs a type define to have element display correct.

type="text"

The standard one line text box. The default type for an INPUT
